# MQTT Server + Node Red + Javascript Vue.js
 This project highlights the use of __*AWS*__ Linux Server with __*Dockerized Mosquitto MQTT Server + Node Red MQTT (Client 1) + MQTT JavaScript (Client 2) + Vite Vue.js*__
